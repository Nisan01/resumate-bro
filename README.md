# 🚀 Resumate Bro — AI-Powered Career Intelligence Platform

Resumate Bro is a full-stack AI-driven web application designed to help job seekers optimize their career journey. From resume analysis to interview preparation and personalized learning paths, the platform provides actionable insights powered by modern AI.

---

## 🎯 Overview

Resumate Bro empowers users to:

* Analyze resumes with ATS-style scoring
* Track and improve professional skills
* Practice interview questions tailored to job roles
* Generate structured career roadmaps
* Monitor progress through a unified dashboard

> Built with a focus on real-world usability, scalability, and clean user experience.

---

## ✨ Features

### 📄 AI Resume Analyzer

* Upload resumes and receive instant ATS score
* Skill match analysis based on job roles
* Suggestions to improve resume quality

### 📊 Skills Tracker

* Track skills and proficiency levels
* Visualize growth over time
* Identify skill gaps

### 🧠 AI Mock Interview Flashcards

* Role-based interview questions
* Adjustable difficulty levels
* Practice-oriented learning experience

### 🛣️ Career Roadmap Generator

* Personalized learning paths for specific roles
* Step-by-step skill development guidance

### 📈 User Dashboard

* Centralized view of user progress
* Resume history tracking
* Clean and intuitive UI

## ⚙️ Getting Started

### Prerequisites

* Node.js installed
* Package manager (npm / yarn / pnpm / bun)

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/resumate-bro.git
cd resumate-bro
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Setup Environment Variables

Create a `.env` file in the root directory and add the following:

```env
JWT_SECRET=your_jwt_secret
DATABASE_URL=your_postgresql_connection_string
OPENAI_API_KEY=your_openai_api_key
OPENAI_PROJECTS_API_KEY=your_openai_projects_api_key
GEMINI_API_KEY=your_gemini_api_key
GROQ_API_KEY=your_groq_api_key_v1
GROQ_API_KEY_V2=your_groq_api_key_v2
GROQ_API_KEY_V3=your_groq_api_key_v3
GOOGLE_CLIENT_ID=your_google_client_id
GOOGLE_CLIENT_SECRET=your_google_client_secret
NEXT_PUBLIC_APP_URL=http://localhost:3000
GOOGLE_REDIRECT_URI=http://localhost:3000/api/auth/google/callback
```

### 4. Run the Development Server

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open your browser and visit:

```
http://localhost:3000
```

Start editing the app by modifying:

```
app/page.tsx
```

The page auto-updates as you make changes.
